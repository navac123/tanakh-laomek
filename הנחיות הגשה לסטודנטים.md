# הנחיות הגשת עבודה — מובןלי: תנ״ך לעומק

## מה מגישים?
בסוף כל עבודה תגישו **שני קבצים**:
1. קובץ **PDF** — לתיעוד ולעיון
2. קובץ **גיבוי (JSON)** — לבדיקה אוטומטית

---

## איך מייצאים את הקבצים מהאפליקציה?

### שלב א׳ — הזינו את שמכם:
1. הגיעו לשלב 5 (קריאה) באפליקציה — הכפתור האחרון בסרגל הניווט
2. בלוח הצד בצד ימין, תמצאו שדה **"שם סטודנט/ית"** — הזינו שם מלא
3. השם יישמר אוטומטית ויופיע בקובץ ההגשה

### שלב ב׳ — ייצוא גיבוי JSON:
1. לחצו על הכפתור **💾 גיבוי JSON להגשה** (בלוח הצד, שלב קריאה)
2. הקובץ יורד אוטומטית בשם: `moovanly_שם_מסמך_תאריך.json`
3. שנו את שם הקובץ ל: `שם_שלכם_מזמור.json`

### שלב ג׳ — ייצוא PDF:
1. לחצו על **ייצוא התוצאה** (כפתור אדום בלוח הצד)
2. בחלון שנפתח, בחרו **"PDF"** או **"הדפסה"**
3. בחרו **"שמור כ-PDF"** (במקום מדפסת)
4. שמרו בשם: `שם_שלכם_מזמור.pdf`

---

## איך מגישים ב-Moodle?
1. היכנסו למטלת ההגשה הרלוונטית בקורס
2. העלו את **שני הקבצים** (PDF + JSON) יחד
3. לחצו שלח

---

## דוגמה לשמות קבצים:
- `יעל_כהן_תהילים13.pdf`
- `יעל_כהן_תהילים13.json`

---

## מה זה בכלל קובץ JSON ולמה אנחנו צריכים אותו?

קובץ ה-JSON הוא **ארגז שבו ארוזה כל העבודה שלכם** — כל הסימונים, הקטגוריות, החבילות, הארגון הויזואלי, הערות המחברת — הכל שמור בו בצורה שמחשב יכול לקרוא.

**למה לא רק PDF?**
ה-PDF יפה לקריאה, אבל קובץ ה-JSON מאפשר למרצה להשתמש בכלי בדיקה שרואה ישירות כמה סימונים עשיתם, כמה קטגוריות יצרתם, כמה חבילות בניתם — ללא צורך לספור ידנית.

**מה יש בתוך הקובץ?**
כל הסימונים שסימנתם על הטקסט, הקטגוריות שיצרתם, החבילות, הארגון בקנבס, והערות המחברת — הכל ארוז יחד.

**האם צריך לפתוח אותו?**
לא! רק להוריד ולהעלות ל-Moodle. אם תפתחו בטעות — תראו שורות קוד. זה תקין, פשוט סגרו.

**האם זה בטוח?**
כן. הקובץ מכיל רק את מה שמילאתם באפליקציה עצמה.

## שאלות נפוצות

**שכחתי להזין שם לפני ייצוא ה-JSON — מה עושים?**
חזרו לשלב קריאה, הזינו שם בשדה "שם סטודנט/ית", ואז לחצו שוב על גיבוי JSON. הקובץ החדש יכיל את השם.

**שכחתי לשמור ויצאתי מהאפליקציה — האם אבדו הנתונים?**
לא! האפליקציה שומרת אוטומטית בדפדפן. פתחו שוב את אותו דפדפן באותו מחשב — הנתונים יחזרו.

**עברתי מחשב / דפדפן — האם העבודה נשמרה?**
האפליקציה שומרת בתוך הדפדפן הספציפי ובמחשב הספציפי. לפני שמחליפים מחשב — הורידו גיבוי JSON ואז טענו אותו מחדש דרך **ייצוא ← ייבוא JSON**.

**הקובץ JSON נראה כמו "קוד" — האם זה תקין?**
כן, זה תקין לחלוטין. אין צורך לפתוח אותו — פשוט העלו כמו שהוא.

**לא מצאתי את כפתור "גיבוי JSON להגשה" — איפה הוא?**
הכפתור נמצא בשלב **קריאה בלבד** (שלב 5 — הכפתור האחרון בסרגל הניווט). ודאו שהגעתם לשלב האחרון.
